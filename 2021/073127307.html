一、Ubuntu的优化

1、软件的卸载

1.1、卸载亚马逊链接

sudo apt-get remove unity-webapps-common

1.2、卸载libreOffice更换为WPS

libreoffice是ubuntu自带的开源office软件，体验效果不好，改为wps

卸载：sudo apt-get remove libreoffice-common

wps的安装：

1、官网https://linux.wps.cn/下载最新版本，选择deb for x64

2、输入sudo dpkg -i wps-office_11.1.0.10161_amd64.deb（根据版本不同命令不同）

3、启动wps后显示“系统缺失字体”

/1 进入C:\Windows\Fonts将所有字体压缩到Fonts压缩包中，将压缩包拷贝到ubuntu中，解压

（直接拷贝字体可能会出错）

/2 将Fonts文件夹在终端中打开，输入 sudo cp * /usr/share/fonts/wps-office

/3 生成字体索引信息 sudo mkfontscale sudo mkfontdir

/4 更新字体缓存 sudo fc-cache

/5 重启wps即可

1.3、谷歌浏览器，搜狗拼音，vscode的安装

参照上一篇文章 《在虚拟机中安装Ubuntu与基本环境的配置》

    exec "w"

3、菜单栏位置

放到底部（终端输入以下）：
gsettings set com.canonical.Unity.Launcher launcher-position Bottom
放到左侧（终端输入以下）：
gsettings set com.canonical.Unity.Launcher launcher-position Left

二、美化Ubuntu

1、主题

1.1、安装unity-tweak-tool:

sudo apt-get install unity-tweak-tool

调整 Unity 桌面环境，还是推荐使用Unity Tweak Tool，这是一个非常好用的 Unity 图形化管理工具，可以修改工作区数量、热区等。

在安装后，输入unity-tweak-tool，遇到一直报错打不开的问题，解决方法：

sudo apt-get install unity-lens-applications 
sudo apt-get install unity-lens-files

1.2 、Flatabulous主题

通过源码安装时，一直报错类似于仓库没有release文件，于是放弃源码安装

若遇到该问题，解决方法：

1、输入命令cd /etc/apt/sources.list.d进入到对应的ppa目录

2、ls看到对应无法下载的list

3、通过sudo rm <....list>删除命令删除

虽然问题解决了，但是主题仍未下载成功.

遂采用deb格式文件下载

wget https://github.com/anmoljagetia/Flatabulous/releases/download/16.04.1/Flatabulous-Theme.deb下载flatabulous主题

sudo dpkg -i Flatabulous-Theme.deb安装

安装完成后，打开unity-tweak-tool软件，进入主题 ，修改为Flatabulous

2、终端

2.1、安装zsh

sudo apt install zsh  #安装zsh
chsh -s /bin/zsh      #将zsh设置默认为shell

2.2、安装oh my zsh

sudo apt install git #git的安装
sh -c $(curl -fsSL https://gitee.com/shmhlsy/oh-my-zsh-install.sh/raw/master/install.sh) #oh my zsh的安装

重启即可

2.3、更换主题

进入theme官网 https://github.com/ohmyzsh/ohmyzsh/wiki/Themes选择你想要的主题，以avit为例

2.4、安装插件

zsh-syntax-highlighting

命令行高亮插件

执行以下命令

git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting

zsh-autosuggestions

命令行命令键入时的历史命令建议插件

执行以下命令

git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions

文件修改

ls -a #查看文件

vim .zshrc #编辑.zshrc配置文件

#输入i进入编辑模式

# 使用的zsh 主题
ZSH_THEME="avit"

# 启用已安装的插件
plugins=(
  git zsh-autosuggestions zsh-syntax-highlighting
)

#esc退出编辑模式，:wq退出vim

即更换好主题和添加好插件

2.5、更换背景

/1、sudo apt-get update 更新系统资源

/2、sudo apt-get install xfce4-terminal 安装xfce4终端

/3、打开xfce4终端

/4、菜单栏编辑->首选项->外观->背景图片选择你想要的壁纸 样式改为拉伸
